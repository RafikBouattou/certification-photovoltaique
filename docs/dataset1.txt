Ce jeu de données contient des informations sur un système de production photovoltaïque. Il est divisé en deux parties principales : les métadonnées et les données chronologiques.

**1. Métadonnées:**

Les métadonnées sont stockées dans le fichier `Metadata/PV generation system metadata.ttl`. Ce fichier utilise la syntaxe Turtle RDF pour décrire les systèmes de production photovoltaïque. Les métadonnées comprennent des informations sur :

*   **Systèmes de production photovoltaïque :** Emplacement, altitude, date de connexion, contractant, puissance nominale, azimut et angle d'inclinaison.
*   **Onduleurs :** Marque, modèle et nombre d'onduleurs.
*   **Panneaux photovoltaïques :** Marque, modèle, nombre de panneaux, dégradation, efficacité et puissance nominale.

**2. Données chronologiques:**

Les données chronologiques sont divisées en deux catégories : les données météorologiques et les données de production photovoltaïque.

*   **Données météorologiques :** Ces données sont stockées dans le répertoire `Time series dataset/Meteorological dataset`. Elles sont ensuite divisées en sous-répertoires pour différents paramètres météorologiques :
    *   Irradiance
    *   Précipitations
    *   Humidité relative
    *   Pression au niveau de la mer
    *   Température
    *   Visibilité
    *   Vent

    Chacun de ces répertoires contient des fichiers CSV avec des données pour chaque année (par exemple, `Irradiance_2021.csv`). Les fichiers CSV ont une colonne `Time` et une colonne pour le paramètre météorologique correspondant. Les données sont enregistrées à un intervalle d'une minute.

*   **Données de production photovoltaïque :** Ces données sont stockées dans le répertoire `Time series dataset/PV generation dataset`. Elles sont divisées en deux sous-répertoires :
    *   `PV stations with panel level optimizer` (Stations PV avec optimiseur au niveau du panneau)
    *   `PV stations without panel level optimizer` (Stations PV sans optimiseur au niveau du panneau)

    **Stations PV avec optimiseur au niveau du panneau**

    Ce répertoire contient deux sous-répertoires : `Inverter level dataset` et `Site level dataset`.

    *   **Inverter level dataset:**
        Ce répertoire contient 44 fichiers CSV, un pour chaque onduleur. Voici quelques exemples :
        *   `Indoor Sports Centre_Inverter.csv`
        *   `Library_Inverter_1.csv`
        *   `LSK North_Inverter.csv`
        *   ...

        Les colonnes de ces fichiers sont les suivantes :
        *   `Time` : Horodatage de la mesure.
        *   `dcVoltage(V)` : Tension en courant continu provenant des panneaux photovoltaïques.
        *   `totalActivePower(W)` : Puissance active totale générée par l'onduleur.
        *   `L1_acCurrent(A)` : Courant alternatif pour la phase 1.
        *   `L1_acFrequency(Hz)` : Fréquence du courant alternatif pour la phase 1.
        *   `L1_acVoltage(V)` : Tension du courant alternatif pour la phase 1.
        *   `L1_activePower(W)` : Puissance active pour la phase 1.
        *   `L1_reactivePower(W)` : Puissance réactive pour la phase 1.
        *   `L2_acCurrent(A)` : Courant alternatif pour la phase 2.
        *   `L2_acFrequency(Hz)` : Fréquence du courant alternatif pour la phase 2.
        *   `L2_acVoltage(V)` : Tension du courant alternatif pour la phase 2.
        *   `L2_activePower(W)` : Puissance active pour la phase 2.
        *   `L2_reactivePower(W)` : Puissance réactive pour la phase 2.
        *   `L3_acCurrent(A)` : Courant alternatif pour la phase 3.
        *   `L3_acFrequency(Hz)` : Fréquence du courant alternatif pour la phase 3.
        *   `L3_acVoltage(V)` : Tension du courant alternatif pour la phase 3.
        *   `L3_activePower(W)` : Puissance active pour la phase 3.
        *   `L3_reactivePower(W)` : Puissance réactive pour la phase 3.

    *   **Site level dataset:**
        Ce répertoire contient 37 fichiers CSV, un pour chaque site. Voici quelques exemples :
        *   `Indoor Sports Centre.csv`
        *   `Library.csv`
        *   `LSK North.csv`
        *   ...

        Les colonnes de ces fichiers sont les suivantes :
        *   `Time` : Horodatage de la mesure.
        *   `generation(kWh)` : Énergie générée en kilowattheures.
        *   `power(W)` : Puissance en watts.

    **Stations PV sans optimiseur au niveau du panneau**

    Ce répertoire contient un sous-répertoire : `Site level dataset`.

    *   **Site level dataset:**
        Ce répertoire contient 23 fichiers CSV, un pour chaque site. Voici quelques exemples :
        *   `CYT (Station 1).csv`
        *   `SQ10.csv`
        *   `Tower A.csv`
        *   ...

        Les colonnes de ces fichiers sont les suivantes :
        *   `Time` : Horodatage de la mesure.
        *   `generation(kWh)` : Énergie générée en kilowattheures.
        *   `power(W)` : Puissance en watts.
